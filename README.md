# Todo List

카테고리, 필터, 진행률 표시, 로컬 저장이 지원되는 순수 HTML/CSS/JS 기반 할 일 목록 웹앱입니다.

## 기능

- 할 일 추가 / 수정 / 삭제
- 완료 여부 체크 (완료 항목은 목록 하단으로 정렬)
- 카테고리 지정 (개인 / 공부 / 업무 / 취미) 및 카테고리별 필터링
- 완료율 진행 바 표시
- `localStorage`를 이용한 새로고침 후에도 데이터 유지
- 반응형 레이아웃 (모바일 화면 대응)

## 사용 방법

`todo-list.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다. 별도의 빌드나 서버 설정이 필요 없습니다.

## 기술 스택

- HTML / CSS / JavaScript (프레임워크 없이 순수 바닐라 구현)
